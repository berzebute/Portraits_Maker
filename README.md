# 🛡️ Portraits Maker

**Portraits Maker**는 사용자가 원하는 이미지를 바탕으로 정교한 초상화(Portraits) 유물을 연성하고 관리할 수 있도록 돕는 Windows 전용 도구입니다. 혼돈스러운 이미지 파일들을 체계적인 초상화 데이터로 변환하십시오.

---

## 🏛️ 주요 기능 (Key Features)

* **정밀한 연성:** 고화질 이미지를 초상화 규격에 맞게 최적화하여 생성합니다.
* **직관적인 인터페이스:** 단순하지만 강력한 사용법을 제공합니다.
* **무설치 배포:** 별도의 런타임 설치 없이 즉시 작동합니다.
* **최적화 해상도 (Resolution):** 게임의 특성에 맞는 최적의 해상도로 저장해줍니다.
* **확장자 자동 변환:** 각 게임에 맞는 확장자를 자동으로 맞춰줍니다.

---

## ⚔️ 지원되는 게임

* EE (BG1, BG2, IWD1)
* Classics D&D Games (BG1, BG2, IWD1, IWD2)
* Pathfinder (Kingmaker, WotR)
* Pillars of Eterniry 1, 2

---

## 🚀 시작하기 (Getting Started)

### 설치 및 실행
1.  [Releases] 페이지에서 최신 버전의 압축 파일을 다운로드합니다.
2.  적절한 폴더에 압축을 해제합니다.
3.  `Portraits Maker.exe`를 실행하여 연성을 시작합니다.
